# Avatar Makeup Beta
Roblox 2025

## Testing Makeup
1) Load MakeupBeta.rbxl in studio.
2) Press Play.
3) Observe makeup applied on avatars
4) Observe Eyebrows and Eyelashes applied on avatars

## Importing Makeup
1) Use supplied sample makeup in MakeupImport Folder
Both FBX and GLTF file types are provided.
2) Copy and paste the imported Makeup Decals into the respective Makeup Folder in each Character.
3) Copy and paste the imported Eyebrow and Eyelash MeshParts into the respective Accessories folder in each Character.
4) Press Play.
5) Observe your imported makeup applied on the chosen character.
6) Use Play/Pause and Reset buttons in top-right corner of viewport to control facial animation in order to observe Eyebrow/Eyelash deformation.
