# Welcome to Roblox Avatar

This repository contains a collection of test materials, early-release and beta downloadables, sample assets, and feature-specific instructions provided for your convenience, experimentation, and evaluation. 

These resources are intended to support testing and exploration of new or evolving functionality.

**Roblox does not commit to maintaining or updating the assets hosted in this GitHub repository**. As features mature and reach full release, their assets and workflows are fully documented and maintained on the [official documentation website](create.roblox.com/docs/avatar). Materials that remain in this repository may become outdated and should be considered reference or experimental content rather than long-term supported resources.
